# A Simple WebAR experience
This is a simple WebAR experience created using AR.js and A-Frame.

Check it out: https://blue-eclipse.github.io/webar/

Or you can host it yourself by cloning this repository and using GitHub Pages. 

Once loaded, provide the site camera permissions and point your device at this marker. 
<img width="512" height="512" alt="default-marker" src="https://github.com/user-attachments/assets/bad1375e-a66a-43d0-a87d-de760ee8f34c" />

Screen brightness and contrast can affect how it looks for your device. Printing the marker should give better consistency for tracking and performance.
